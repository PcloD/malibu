Class: Timer

Timer

Constructor

new Timer(autostart, autoupdate)

A timer utility used to invoke function once or repeatedly in time.
Parameters:
Name Type Description
autostart Boolean if true the Timer will start counting time immediately. Otherwise start() needs to be called manually later.
autoupdate Boolean if true the Timer will be updating on every frame. Otherwise, update() needs to be called on every frame.

onEvery(interval, time, callback, repeat) → {Object}

Invokes the callback repeatedly overtime. All time values in ms.
Parameters:
Name Type Description
interval Number how often to invoked the function. It can be an array of two elements specyfing a min/max range
time Number when to start (i.e. delay, counted from 'now' i.e from when this method is called)
callback Number the callback to be invoked
repeat Number how many times to repeat. If ommited or -1 will repeat infinitely 0 will never invoke the function (in fact it won't even be added)
