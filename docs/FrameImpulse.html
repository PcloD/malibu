FrameImpulse

A utility to handle requestAnimationFrame loops.
Source:
  • FrameImpulse.js, line 1

Example

var render = function() {
  // Do some rendering logic in here
}

// When the loop needs to be activated
FrameImpulse.on(render);

// ...and when it needs to stop
FrameImpulse.off(render);

Methods

(static) off(callback)

Removes a listener to be called on every frame
Parameters:
Name Type Description
callback function the function used as callback for the listener. Needs to be the same function as passed to the on() when it was being registered.
