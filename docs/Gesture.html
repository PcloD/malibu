Class: Gesture

Gesture

Constructor

new Gesture(optionsopt)

Possible optins include:
  • maxTime - how long before swipe is not considered a swipe (default 300ms)
  • minDistance - how much must the user move to consider this a swipe (default 30px)
  • tolerance - in radians, how far off vertical or horizontal axis is considered as swipe default: 0.1, don't make it larger than 0.25 (i.e. 45deg)
Parameters:
Name Type Attributes Description
options Object <optional>
 object holding settings (see above)
