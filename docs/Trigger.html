Class: Trigger

Trigger

Constructor

new Trigger()

Trigger is a simple utility used to create events.

A Trigger can only send one type of event and it will always notify all of its listeners.

In order to build a more robust event system, use multiple Trigger objects as properties.

Examples
// Simple usage for a single type of event
var menuPress = new Trigger();

var menuPressListener = function(params) {
  console.log("menuPress trigger fired");
  // params = whatever was passed to the trigger method (see below)
  console.log(params);
}

menuPress.on(menuPressListener);

// ... then somewhere in the code:
menuPress.trigger({ id: 5 });

// ... and eventually
menuPress.off(menuPressListener);

	
// If there are multiple events to handle, 
// simply create multiple triggers
var car = {
  engineStarted: new Trigger(),
  brakeApplied: new Trigger(),
  gearChanged: new Trigger()
}

Methods

off(callback)

Removes a listener from this trigger.

If the passed callback is not a listener of this trigger, this function will not throw any warnings, it will just return.

If this function is called from within a function that is a listener for that trigger, the callback will not be removed until all other listeners are called.

Parameters:
Name Type Description
callback function the function used as callback for the listener. Needs to be the same function as passed to the on() when it was being registered.

trigger(dataopt)

Fires this trigger passing data as srgument to all listeners.

If this function is called from within a function that is a listener for that trigger, the trigger will not be fired until all other listeners are called for the previous one.

Parameters:
Name Type Attributes Description
data Object <optional>
 An object specifying the events parameters. All listeners will receive this object as argument.
